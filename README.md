# Laptop Webpage Project

This project is a simple webpage dedicated to showcasing various laptop products. It includes detailed information about each laptop, including specifications, images, and pricing.

## Project Structure

```
laptop-webpage
├── src
│   ├── index.html          # Main entry point for the webpage
│   ├── product.html        # Detailed view of a specific laptop
│   ├── css
│   │   └── styles.css      # Styles for the webpage
│   ├── js
│   │   └── main.js         # JavaScript functionality
│   ├── components
│   │   ├── header.html     # Header component with navigation
│   │   └── footer.html     # Footer component with copyright info
│   └── data
│       └── laptops.json    # JSON data for laptop products
├── .gitignore              # Files to ignore in version control
├── package.json            # npm configuration file
└── README.md               # Project documentation
```

## Features

- Responsive design for various devices
- Dynamic loading of laptop data from JSON
- Interactive elements for better user experience

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd laptop-webpage
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Open `src/index.html` in your browser to view the webpage.

## Usage

- Navigate through the homepage to view available laptops.
- Click on a laptop to see detailed information on the product page.
